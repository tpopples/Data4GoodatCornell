Data4Good at Cornell is a student-led initiative applying data science to public-interest and policy problems.

We build reproducible analyses, run applied workshops, and collaborate with community partners to translate data into actionable insights.

What This Repository Contains
	•	Workshop materials and session notebooks
	•	Reproducible data pipelines
	•	Applied project code
	•	Sample datasets (processed only; raw data not tracked)

All work emphasizes:
	•	Transparency
	•	Reproducibility
	•	Clear documentation
	•	Ethical data use

Tech Stack
Python · Jupyter · Census/ACS · Geospatial tools · Git/GitHub
